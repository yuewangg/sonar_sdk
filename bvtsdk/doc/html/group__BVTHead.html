Detailed Description

A head consists of a group of co-planar transducers which are operated simultaneously to produce (ultimately) a single 2d image. The Head object provides functions to change the range window as well as produce pings.

Latest-generation BlueView sonar support two modes of operation, a standard and an alternate mode. The alternate mode provides improved imagery for wide field-of-view sonar (greater than 45 degrees) at the cost of a slower max ping rate, while dual mode maximizes ping rate but reduces image quality. This call provides the user details about whether the alternate ping mode is supported on the given sonar head.

Set the image processing resolution. The RES_AUTO setting is highly recommended, as it adapts via a formula according to the stop range, whereas the other ranges are fixed values, and should only be used in specialized cases, such as requesting high resolution for longer distances (which will increase the processing time required to create the image). R-Theta images may use either this function or BVTHead_SetRangeResolution(), depending on the degree of control required.
