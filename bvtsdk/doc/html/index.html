This manual will guide you through the installation and use of this of the BlueView Sonar Development Kit (SDK). Our goal is to get you started on the image acquisition portion of your sonar imaging project, and to help you develop a successful application.

General Overview

This SDK includes an API that allows you to communicate with a BlueView sonar. The API provides functions to acquire pings, produce images, as well as read from and write to the BlueView .son file format. Sample data in the .son file format is provided.

System Requirements

While the final system requirements depend greatly on the target application, we recommend the following minimum requirements to ensure that the sonar and the SDK perform acceptably.

  • Windows: XP or higher
  • Linux: (Tested on Ubuntu 12 for both x86 and x86-64, and buildroot 2013.05-based ARMv7)
  • OS X: 10.7+ (64-bit Intel CPUs only, no PowerPC support)
  • x86 (32-bit) CPUs must support SSE2
  • 750MHz or faster CPU
  • 512MB or more of RAM
  • 200MB or more of free disk space
  • Ethernet port (100 Mbit/s required)

Installation

Windows

To install the SDK, simply run setup.exe from the SDK directory on the distribution CD. Please see the Getting Started section for more information. You can also copy the installed directory to a directory of your choice if you want to develop against multiple versions of the SDK at once.

Linux and OS X

Simply extract the provided archive (tar/zip) to a directory of your choice.
