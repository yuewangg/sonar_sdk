Detailed Description

Store a color image. The API is nearly identical to MagImage. The main difference is the pixel datatype. In ColorImage, each pixel is a single unsigned int.

  • Byte 0: Red Value
  • Byte 1: Green Value
  • Byte 2: Blue Value
  • Byte 3: Alpha Value
