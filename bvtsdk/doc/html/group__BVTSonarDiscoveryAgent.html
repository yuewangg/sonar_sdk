RetVal BVTSonarDiscoveryAgent_GetSonarInfo ( BVTSonarDiscoveryAgent  self,
int  sonar_index,
char *  buffer,
int  buffer_size

Returns the host IP address (as a null-terminated string) for the specified sonar. The sonar_index parameter is zero-based, that is, if BVTSonarDiscoveryAgent_GetSonarCount returned a value of 3, then valid sonar_index values are 0, 1, and 2. A buffer size of 20 characters should be sufficient for the buffer.
