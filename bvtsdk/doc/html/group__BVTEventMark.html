Detailed Description

An EventMark allows the user to insert arbitrary information into a sonar file. The user can store two strings, one called a "Key" string and the other a "Text" string, in an EventMark. The EventMarks are stored and retrieved independently of Heads or Pings. NOTE: EventMark objects will only be saved to a sonar of type FILE.
